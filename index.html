Project in Console

Visual Studio Code is a HTML coding platform where people can code in a user-friendly app. You can easily download it then create a folder, open it in VSC, and then start coding.

To get to the Console Screen, you have to right click on the web page, and select inspect and then a pop-up will apear. After that, click the tab "console", and all the messeges you printed in the code will show. It is used to show messeges from code and you can easily print anything by using the "console.log()".
